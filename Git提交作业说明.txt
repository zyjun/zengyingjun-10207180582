1.首先安装git bash

2.用windows自带的命令行窗口输入：“git --version”
  如果有出现版本说明(如git version 1.7.11.msysgit.1)，则说明你的环境变量path已配置  好;否则,则需配置path。如果你的Git是安装在D盘目录下，则在path变量下添加(注意：是    添加，不是更改)配置路径:   D:\Git\cmd

3.此时，再在cmd运行“git --version”是否可以了(否则自己检查路径是否配置正确)

4.以上弄好后，则选择一个路径放置你将要同步的文件夹(10_18_homework)。如若选择放在E  盘下，则在命令窗口下输入：“E: ”并回车。

5.运行“ git clone https://github.com/summit2013/10_18_homework.git”，此过程会慢，因为会下载所有其他人已上传的作业。此时会在E盘下生成一个文件夹10_18_homework，该文件夹即是我们班交作业的库。

6.在10_18_homework文件夹下添加你自己的文件夹

7.再将你做好的作业放到你自己的文件夹下。

8.运行 “cd 10_18_homework”（确保你已运行过“E:”即第四步运行过的步骤）

9.运行“git add -A”

10.运行“git commit -m "说明"  (注：说明可以是任意的，如对提交作业的说明,一般为"update"）

11.运行“git push”

12.运行过11后，会要求你输入用户名，此时输入你自己创建的那个库的帐户名并回车：如：  
 
13.运行过12后，会要求你输入密码，输入你的帐户名的密码并回车。

经过已上步骤即上交了。

   注：在每次提交之前请确保你的本地库与远程库文件是相同的。可运行：“git pull”将       你本地库与远程库同步。
 
      此后每次添加作业提交便可从第9步运行，如果是更改作业便可从第10步运行。
